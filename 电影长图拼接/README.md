# 🎞️ CineStitcher Pro | 电影台词长图大师

一个极简、专业的电影台词长图制作工具。无需下载安装，通过浏览器即可将视频瞬间拼接为极具电影感的长图。

### ✨ 核心功能
- **实时捕捉**：播放视频时按下 `Space` 键即可精准捕捉当前台词帧。
- **双向联动**：管理列表与预览区域实时高亮联动，精准定位每一帧。
- **智能裁剪**：支持 Top/Bottom 裁剪，完美剔除视频 Logo 及进度条 UI。
- **色彩增强**：内置电影感色彩增强滤镜，让截图更饱满。
- **无损导出**：支持原画级分辨率拼接导出。

### 🚀 如何使用
1. 上传本地视频文件。
2. 看到台词时点击“捕捉”或按空格。
3. 鼠标在右侧画布上下拖动调整台词间距。
4. 调整裁剪滑块避开视频水印。
5. 点击“导出超清长图”。
