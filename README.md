# 🎞️ CineStitcher Pro | 影拼大师

**CineStitcher Pro** 是一款专为影视爱好者、创作者设计的「电影台词长图」制作工具。它完全基于浏览器运行，无需下载安装，通过极简的交互逻辑，帮助你快速将视频精彩片段拼接为极具质感的台词长图。

---

## ✨ 核心亮点

### 1. 🎬 丝滑的捕捉体验
- **空格键快速捕捉**：播放视频时，只需按下 `Space` 键，即可瞬间捕捉当前画面帧。
- **高效素材管理**：左侧帧管理器实时生成缩略图，支持一键删除误点帧。

### 2. 🔗 独创的双向高亮联动
- **正向联动**：鼠标划过左侧列表，右侧预览区对应切片立即高亮并勾勒边框。
- **反向定位**：鼠标在长图预览区滑动，左侧列表自动滚动并锁定对应帧，实现像素级精准校验。

### 3. ✂️ 智能画幅裁剪
- **双向裁剪系统**：内置 `Top Crop` 与 `Bottom Crop` 滑块，一键剔除视频水印、Logo、进度条及播放器 UI。
- **全局同步**：只需调整一次滑块，所有台词条同步对齐。

### 4. 🎨 电影感视觉增强
- **自由间距调节**：通过在画布上直接拖拽，自由控制台词显示的宽度与留白。
- **色彩饱和度增强**：内置对比度与饱和度算法，让导出的长图比视频原截图更具艺术感。
- **无损高清导出**：支持原视频分辨率级别的高清 PNG 导出。

---

## 🚀 快速开始

1. **上传视频**：点击侧边栏顶部的“选择文件”，载入你的本地视频。
2. **捕捉精彩**：播放视频，在出现台词的瞬间点击“捕捉台词帧”或按 `Space` 键。
3. **精修布局**：
   - 在右侧画布上**上下拖拽**鼠标，调整台词的显示高度。
   - 调节**顶部/底部裁剪**滑块，隐藏多余的水印和黑边。
4. **校对检查**：移动鼠标查看预览区的高亮反馈，确保每一句台词都精准对齐。
5. **一键导出**：点击“导出超清长图”，保存你的作品。
